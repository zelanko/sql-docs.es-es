# <a name="mining-structure-view-data-mining-model-designer"></a>Estructura de minería de datos (vista del Diseñador de modelos de minería)

 La vista **Estructura de minería de datos** incluye las herramientas y los cuadros de diálogo siguientes:  
  
-   [Nuevo cuadro de diálogo de modelo de minería de datos &#40;vista de la estructura de minería de datos&#41;](new-mining-model-dialog-box-mining-structure-view.md)  
  
-   [Seleccione una columna del cuadro de diálogo &#40;vista de la estructura de minería de datos&#41;](select-a-column-dialog-box-mining-structure-view.md)  
  
-   [Seleccione un cuadro de diálogo de la columna de clave de tabla anidada &#40;vista de la estructura de minería de datos&#41;](select-a-nested-table-key-column-dialog-box-mining-structure-view.md)
