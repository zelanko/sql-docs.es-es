# <a name="translation-details-translations-tab-cube-designer-analysis-services---multidimensional-data"></a>Detalles de traducción (pestaña Traducciones, Diseñador de cubos) (Analysis Services - Datos multidimensionales)
  Use el panel **Detalles de transacción** de la pestaña **Transacciones** en el Diseñador de cubos para definir y administrar traducciones en el cubo seleccionado.  
  
## <a name="options"></a>Opciones  
 **Idioma predeterminado**  
 Establece los nombres de los objetos de cubo en el idioma predeterminado.  
  
 **Tipo de objeto**  
 Muestra la propiedad que se va a traducir. Es posible traducir las siguientes propiedades:  
  
-   Grupo de medida  
  
     `Caption`propiedad  
  
-   Measure  
  
     Propiedades `Caption` y `DisplayFolder`  
  
-   Dimensión  
  
     `Caption`propiedad  
  
-   Perspectiva  
  
     `Caption`propiedad  
  
-   Indicador clave de rendimiento (KPI)  
  
     Propiedades `Caption`, `Description` y `DisplayFolder`.  
  
-   Acción  
  
     `Caption`propiedad  
  
-   Conjunto con nombre  
  
     `Caption`propiedad  
  
-   miembro calculado  
  
     `Caption`propiedad  
  
 **\<>de idioma**  
 Escriba el valor de la propiedad del objeto de cubo en el idioma seleccionado.  
  
## <a name="context-menu"></a>Menú contextual  
 Las siguientes opciones están disponibles en el menú contextual, que se muestra al hacer clic con el botón derecho en una traducción del panel **Detalles de traducción** :  
  
|Término|Definición|  
|----------|----------------|  
|**Nueva traducción**|Seleccione esta opción para mostrar el cuadro de diálogo **Seleccionar idioma** y crear una nueva traducción. La traducción se mostrará como una nueva columna en la cuadrícula **Detalles de traducción** .|  
|**Eliminar traducción**|Seleccione esta opción para eliminar la traducción seleccionada.|
