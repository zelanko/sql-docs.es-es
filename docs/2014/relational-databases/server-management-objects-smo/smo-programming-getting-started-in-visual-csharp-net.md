# <a name="getting-started-in-visual-c-net"></a>Introducción a Visual C# .NET
  [!INCLUDE[csprcs](../../includes/csprcs-md.md)] .NET es uno de los lenguajes de programación administrados compatibles con common language runtime (CLR) en [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)] . NET. Para obtener más información sobre el lenguaje C#, vea el [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)] documentación de. NET. Puede utilizar [!INCLUDE[vsprvslong](../../includes/vsprvslong-md.md)] y [!INCLUDE[vsOrcas](../../includes/vsorcas-md.md)] para desarrollar aplicaciones SMO.  
  
## <a name="references-and-imports"></a>Referencias e importaciones  
 Una vez haya iniciado el proyecto de [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)], debe agregar las referencias a los ensamblados SMO. A continuación, importe los espacios de nombres necesarios para que su programa pueda reconocer los tipos SMO.  
  
 En [!INCLUDE[ssCurrent](../../includes/sscurrent-md.md)], los ensamblados SMO se encuentran en la carpeta [!INCLUDE[ssSampPathSDK](../../includes/sssamppathsdk-md.md)] .
