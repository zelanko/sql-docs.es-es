# <a name="cinstance-class"></a>Clase CInstance
  La [clase CInstance](cinstance-class.md) representa una instancia de un [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] cliente.  
  
 Esta clase puede utilizarse para administrar los protocolos y las bibliotecas de red asociados a la instancia de un cliente de [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)].
