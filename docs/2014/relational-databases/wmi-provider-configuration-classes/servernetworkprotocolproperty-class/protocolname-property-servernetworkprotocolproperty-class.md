# <a name="protocolname-property-servernetworkprotocolproperty-class"></a>Propiedad ProtocolName (clase ServerNetworkProtocolProperty)
  Obtiene el nombre del protocolo que posee la propiedad a la que se hace referencia.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
object  
.PropertyName [= value]  
```  
  
## <a name="parts"></a>Partes  
 *object*  
 Un [clase ServerNetworkProtocolProperty](servernetworkprotocolproperty-class.md) objeto que representa un atributo del protocolo de red en la instancia de [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].  
  
## <a name="property-valuereturn-value"></a>Valor de propiedad y valor devuelto  
 Valor de cadena que especifica el nombre del protocolo que posee la propiedad.
