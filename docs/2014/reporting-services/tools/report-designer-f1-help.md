# <a name="report-designer-f1-help"></a>Diseñador de informes (Ayuda F1)
  En esta sección se proporciona ayuda F1 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] [!INCLUDE[ssRSnoversion](../../includes/ssrsnoversion-md.md)] para los asistentes, vistas y cuadros de diálogo de diseñador de informes. También puede abrir la Ayuda F1 mientras trabaja en un cuadro de diálogo; para ello, presione F1 o haga clic en el botón Ayuda que se muestra en el cuadro de diálogo.
