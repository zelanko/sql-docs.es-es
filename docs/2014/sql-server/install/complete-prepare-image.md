# <a name="complete---prepare-image"></a>Completar - Preparar imagen
  Use la página **Completar** para comprobar que la preparación de la imagen se ha completado.  
  
## <a name="options"></a>Opciones  
 Este cuadro de diálogo incluye un vínculo al archivo de registro del programa de instalación para esta operación. Para completar la preparación de la imagen, haga clic en **Cerrar**.
