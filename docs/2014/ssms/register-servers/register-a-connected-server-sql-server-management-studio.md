# <a name="register-a-connected-server-sql-server-management-studio"></a>Registrar un servidor conectado (SQL Server Management Studio)
  En este tema se describe cómo registrar servidores en [!INCLUDE[ssCurrent](../../includes/sscurrent-md.md)] mediante [!INCLUDE[ssManStudioFull](../../includes/ssmanstudiofull-md.md)]. Mediante el registro del servidor, puede guardar la información de conexión correspondiente a los servidores a los que accede con frecuencia. Un servidor puede registrarse antes de conectarlo o cuando se conecta desde el Explorador de objetos.  
  
 **En este tema**  
  
-   **Para registrar un servidor, utilizando:**  
  
     [SQL Server Management Studio](#SSMSProcedure)  
  
##  <a name="SSMSProcedure"></a> Usar SQL Server Management Studio  
  
#### <a name="to-register-a-connected-server"></a>Para registrar un servidor conectado  
  
1.  En el Explorador de objetos, haga clic con el botón derecho en un servidor al que ya esté conectado y haga clic en **Registrar**.  
  
     **Nombre del servidor**  
     Escriba el nombre que desea utilizar para el servidor registrado. El registro de un servidor local o remoto con [!INCLUDE[ssManStudioFull](../../includes/ssmanstudiofull-md.md)] permite almacenar la información de conexión del servidor para futuras conexiones. Este campo muestra de manera predeterminada el nombre de servidor especificado en la conexión. Puede conservar este nombre o indicar otro nombre de uso sencillo.  
  
     **Descripción del servidor**  
     Escriba una descripción opcional del servidor. El número máximo de caracteres permitido es 250.  
  
     **Seleccione un grupo de servidores**  
     Seleccione el grupo de servidores donde desea guardar el registro del servidor.  
  
     **Nuevo grupo**  
     Haga clic en este botón para abrir el cuadro de diálogo **Nuevo grupo** , donde puede crear un grupo de servidores para el servidor registrado.  
  
     **Guardar**  
     Haga clic para guardar la información insertada y crear un servidor registrado.
