# <a name="sql-editor-visual-database-tools"></a>Editor SQL (Visual Database Tools)
  Utilice el Editor SQL para editar procedimientos almacenados, funciones, desencadenadores y scripts SQL existentes. Esta ventana aparece al abrir cualquiera de estos objetos. Si desea crear una instrucción SQL nueva para ejecutarla en el origen de datos, use el [panel SQL](visual-database-tools.md) del Diseñador de consultas.  
  
 El editor SQL proporciona numerosas características de edición de texto SQL de gran utilidad, entre las que se incluyen las siguientes:  
  
-   Codificación en colores de palabras clave SQL para minimizar los errores de ortografía y de sintaxis.  
  
-   Generación de esquemas de procedimientos almacenados y desencadenadores.  
  
-   Funciones de edición útiles, incluidas las operaciones de cortar, copiar, pegar y arrastrar.  
  
-   Cambio del comportamiento del editor (mediante la selección de **Opciones** en el menú **Herramientas** ) para modificar los espacios virtuales, el ajuste de línea, los números de línea y el tamaño de tabulación.  
  
-   Ayuda para administrar puntos de interrupción de depuración.  
  
> [!NOTE]  
>  IntelliSense no está disponible en el Editor SQL.  
  
 Al editar instrucciones SQL, algunas instrucciones Transact-SQL se incluyen en un cuadro rodeado por una línea fina. Esto ayuda a dividir visualmente el código SQL en secciones de comandos e identifica bloques de instrucciones SQL que pueden diseñarse de forma gráfica mediante el Diseñador de consultas. Para más información sobre cómo usar el Diseñador de consultas, vea [Temas de procedimientos de diseño de consultas y vistas &#40;Visual Database Tools&#41;](design-queries-and-views-how-to-topics-visual-database-tools.md).
