Para ver las propiedades del servicio, en el panel de detalles, haga clic con el botón derecho en el servicio y, después, haga clic en **Propiedades**.  
  
 Para obtener información acerca de [!INCLUDE[ssDEnoversion](../../includes/ssdenoversion-md.md)], vea los Libros en pantalla.
