# <a name="task-7-viewing-updates-made-using-master-data-manager-in-excel"></a>Tarea 7: ver las actualizaciones realizadas con Master Data Manager en Excel
  En esta tarea, comprobará que puede ver los datos actualizados desde MDS en Excel.  
  
1.  Ahora, cambie a la ventana de excel que tenga **Cleansed and Matched Suppliers.xls** hoja de cálculo abierta.  
  
2.  Haga clic en **actualizar** botón en la cinta de opciones.  
  
     ![Excel - botón Actualizar en la pestaña de datos maestros](../../2014/tutorials/media/et-viewupdatesmadeusingmdminexcel-01.jpg "Excel - botón Actualizar en la pestaña de datos maestros")  
  
3.  Tenga en cuenta que **nombres** (California, Nueva York etc.) se muestran para el **estado** campo junto con sus **códigos**.  
  
     ![Excel - actualizar códigos de estado y los nombres de](../../2014/tutorials/media/et-viewupdatesmadeusingmdminexcel-02.jpg "Excel - actualizado los nombres y códigos de estado")
