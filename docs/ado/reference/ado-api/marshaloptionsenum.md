# <a name="marshaloptionsenum"></a>MarshalOptionsEnum
Especifica qué registros se deben devolver al servidor.  
  
|Constante|Valor|Descripción|  
|--------------|-----------|-----------------|  
|**adMarshalAll**|0|Predeterminado: Devuelve todas las filas en el servidor.|  
|**adMarshalModifiedOnly**|1|Devuelve solo las filas al servidor modificadas.|  
  
## <a name="adowfc-equivalent"></a>Equivalente de ADO y WFC  
 Paquete: **com.ms.wfc.data**

|--------------|  
|AdoEnums.MarshalOptions.ALL|  
|AdoEnums.MarshalOptions.MODIFIEDONLY|
