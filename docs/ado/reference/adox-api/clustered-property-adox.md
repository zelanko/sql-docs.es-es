# <a name="clustered-property-adox"></a>Clustered (propiedad, ADOX)
Indica si el índice está agrupado.  
  
## <a name="settings-and-return-values"></a>Configuración y valores devueltos  
 Establece y devuelve un **booleano** valor.  
  
## <a name="remarks"></a>Comentarios  
 El valor predeterminado es **False**.  
  
 Esta propiedad es de solo lectura en [índice](../../../ado/reference/adox-api/index-object-adox.md) objetos que ya se han anexado a una colección.
