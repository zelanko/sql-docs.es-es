# <a name="extended-events-for-python"></a>Eventos extendidos para Python

Esta sección está en desarrollo.

> [!NOTE]
> Compatibilidad con Python es una característica nueva de SQL Server 2017 y está en versión preliminar. Busque información más pronto.

En general, puede supervisar los scripts externos, incluida una que se ejecutase Python, utilizando el mismo marco de trabajo que se proporcionó para la ejecución de script de R en SQL Server 2016. Para obtener más información, consulte [Extended Events para R Services](../r/extended-events-for-sql-server-r-services.md).
