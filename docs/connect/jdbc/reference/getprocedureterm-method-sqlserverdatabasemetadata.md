# <a name="getprocedureterm-method-sqlserverdatabasemetadata"></a>Método getProcedureTerm (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera el término preferido para "procedimiento" en esta base de datos.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public java.lang.String getProcedureTerm()  
```  
  
## <a name="return-value"></a>Valor devuelto  
 Un objeto **String** que contiene el término para el procedimiento.  
  
## <a name="exceptions"></a>Excepciones  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Notas  
 Este método getProcedureTerm especificado por el método getProcedureTerm en la interfaz java.sql.DatabaseMetaData.
