# <a name="getshort-method-sqlservercallablestatement"></a>Método getShort (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera el valor del parámetro designado como **short** en el lenguaje de programación Java.  
  
## <a name="overload-list"></a>Lista de sobrecargas  
  
|NOMBRE|Descripción|  
|----------|-----------------|  
|[getShort (int)](../../../connect/jdbc/reference/getshort-method-int.md)|Recupera el valor del parámetro designado como un objeto **short** en el lenguaje de programación Java según el índice de parámetro.|  
|[getShort (java.lang.String)](../../../connect/jdbc/reference/getshort-method-java-lang-string.md)|Recupera el valor del parámetro designado como un objeto **short** en el lenguaje de programación Java según el nombre del parámetro.|
