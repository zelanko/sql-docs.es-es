# <a name="length-method-sqlserverclob"></a>Método length (SQLServerClob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Devuelve el número de caracteres del objeto CLOB.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public long length()  
```  
  
## <a name="return-value"></a>Valor devuelto  
 La longitud del CLOB en número de caracteres.  
  
## <a name="exceptions"></a>Excepciones  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Observaciones  
 El método length especifica este método length en la interfaz java.sql.Clob.
