# <a name="position-method-sqlservernclob"></a>Método position (SQLServerNClob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera la posición del carácter del elemento especificado **NClob** un objeto o una subcadena en la **NClob** en función de la posición inicial especificada.  
  
## <a name="overload-list"></a>Lista de sobrecargas  
  
|Nombre|Description|  
|----------|-----------------|  
|[Método Position &#40;java.sql.NClob, long&#41;](../../../connect/jdbc/reference/position-method-java-sql-nclob-long.md)|Recupera la posición del carácter en el que el especificado **NClob** objeto *searchstr* aparece en este **NClob** objeto.|  
|[Método Position &#40;java.lang.String, long&#41; &#40;SQLServerNClob&#41;](../../../connect/jdbc/reference/position-method-java-lang-string-long-sqlservernclob.md)|Recupera la posición de carácter en la que la subcadena especificada *searchstr* aparece en la **NCLOB** valor representado por este **NClob** objeto.|
