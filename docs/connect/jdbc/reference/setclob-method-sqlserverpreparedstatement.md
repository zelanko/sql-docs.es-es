# <a name="setclob-method-sqlserverpreparedstatement"></a>Método setClob (SQLServerPreparedStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Establece el parámetro designado en un objeto java.sql.Clob.

|Nombre|Description|  
|----------|-----------------|  
|[setClob método &#40; int, java.io.Reader &#41;](../../../connect/jdbc/reference/setclob-method-int-java-io-reader.md)|Establece el parámetro designado en un objeto java.sql.Clob.|  
|[setClob método &#40; int, java.sql.Clob &#41;](../../../connect/jdbc/reference/setclob-method-int-java-sql-clob.md)|Establece el parámetro designado para el objeto java.SQL.DatabaseMetaData determinado.|
