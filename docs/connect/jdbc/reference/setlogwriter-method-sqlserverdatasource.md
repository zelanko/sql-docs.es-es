# <a name="setlogwriter-method-sqlserverdatasource"></a>método setLogWriter (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Este método solo se utiliza de forma interna. Para obtener más información acerca del registro, consulte [funcionamiento del controlador de seguimiento](../../../connect/jdbc/tracing-driver-operation.md).  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public void setLogWriter(java.io.PrintWriter out)  
```  
  
#### <a name="parameters"></a>Parámetros  
 *out*  
  
 Un objeto PrintWriter.  
  
## <a name="remarks"></a>Comentarios  
 Este método setLogWriter especificado por el método setLogWriter en la interfaz javax.sql.DataSource.
