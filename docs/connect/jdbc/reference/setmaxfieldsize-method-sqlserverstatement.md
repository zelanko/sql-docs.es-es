# <a name="setmaxfieldsize-method-sqlserverstatement"></a>setMaxFieldSize (método) (SQLServerStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Establece el límite para el número máximo de bytes en un [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) columna almacena valores de caracteres o binarios para el número de bytes determinado.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public final void setMaxFieldSize(int max)  
```  
  
#### <a name="parameters"></a>Parámetros  
 *max*  
  
 Un **int** que indica el número máximo de bytes.  
  
## <a name="exceptions"></a>Excepciones  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Comentarios  
 Este método setMaxFieldSize especificado por el método setMaxFieldSize en la interfaz java.sql.Statement.
