# <a name="setshort-method-sqlserverpreparedstatement"></a>Método setShort (SQLServerPreparedStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Establece el parámetro designado en el valor **short** determinado.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public final void setShort(int index,  
                           short x)  
```  
  
#### <a name="parameters"></a>Parámetros  
 *index*  
  
 Valor **int** que indica el número de parámetro.  
  
 *x*  
  
 Un **corto** valor.  
  
## <a name="exceptions"></a>Excepciones  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Notas  
 Este método setShort especifica este método setShort en la interfaz java.sql.PreparedStatement.
