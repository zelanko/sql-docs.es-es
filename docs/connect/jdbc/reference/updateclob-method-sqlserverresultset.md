# <a name="updateclob-method-sqlserverresultset"></a>Método updateClob (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Actualiza la columna designada con un valor java.sql.Clob.

|[updateClob método &#40; int, java.io.Reader &#41;](../../../connect/jdbc/reference/updateclob-method-int-java-io-reader.md)|Actualiza la columna designada mediante el objeto Reader especificado.|
