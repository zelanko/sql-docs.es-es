# <a name="finding-more-ole-db-driver-for-sql-server-information"></a>Buscar más OLE DB controlador para información sobre SQL Server
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-asdbmi-md](../../includes/appliesto-ss-asdb-asdw-pdw-asdbmi-md.md)]

[!INCLUDE[Driver_OLEDB_Download](../../includes/driver_oledb_download.md)]

  Para obtener más información sobre el controlador OLE DB para SQL Server y otros tipos de programación del acceso a datos, vea los siguientes recursos en línea:  
  
 [Acceso a datos y el centro de desarrolladores de almacenamiento](http://go.microsoft.com/fwlink?linkid=4173)  
 Este sitio proporciona artículos técnicos, ejemplos de código, y tecnologías de acceso a otros recursos para el controlador OLE DB para SQL Server y otros datos.  
  
 [Blog de acceso de datos](http://go.microsoft.com/fwlink/?LinkId=48617)  
 Este blog proporciona información sobre las tecnologías de acceso a datos y le permite interactuar directamente con miembros del equipo del producto.  
  
 [Data Access Components para Windows](http://go.microsoft.com/fwlink/?LinkId=107907)  
 Este sitio incluye la documentación más reciente del SDK de Data Access Components para Windows (DAC para Windows). DAC para Windows es la versión de los componentes de acceso a datos (denominada MDAC en versiones anteriores) que se incluye en Windows Vista y Windows Server 2008.  
  
 El SDK de DAC para Windows incluye la documentación de Objetos de datos ActiveX (ADO), OLE DB y Conectividad abierta de bases de datos (ODBC).
