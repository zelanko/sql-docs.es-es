# <a name="manage-a-cdc-service"></a>Administrar un servicio CDC
  Puede usar la Consola del diseñador CDC para ver los servicios que creó con la Consola de configuración del servicio CDC y administrar todas las instancias del servicio CDC de Oracle.  
  
 Haga clic en el nombre del servicio en el panel izquierdo para mostrar información sobre el servicio y para administrarlo.  
  
> [!NOTE]  
>  Primero debe crear un servicio con la Consola de configuración del servicio CDC para agregar servicios a esta lista. Para obtener información acerca de cómo crear un servicio, vea la Ayuda en pantalla proporcionada con la Consola de configuración del servicio CDC.  
  
## <a name="what-you-can-do-when-you-display-the-cdc-service-information"></a>Qué puede hacer cuando se muestra información del servicio CDC  
 Puede hacer lo siguiente cuando se muestra información sobre un servicio:  
  
 **Crear una nueva instancia de CDC para el servicio seleccionado**  
  
 Haga clic en **Nueva instancia CDC de Oracle** en el panel derecho para crear una instancia nueva para ese servicio. El Asistente para nueva instancia se abrirá para crear la instancia. Para obtener más información acerca del Asistente para nueva instancia, vea [Use the New Instance Wizard](../../integration-services/change-data-capture/use-the-new-instance-wizard.md).  
  
 **Iniciar todas las instancias CDC del servicio**  
  
 Haga clic en **Iniciar todas las instancias** en el panel derecho para empezar a capturar datos de todas las instancias definidas en el servicio.  
  
 **Detener todas las instancias CDC del servicio**  
  
 Haga clic en **Detener todas las instancias** para detener la captura de datos modificados para todas las instancias del servicio.
