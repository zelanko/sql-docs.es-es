# <a name="connect-to-data-sources-with-the-sql-server-import-and-export-wizard"></a>Conectarse a orígenes de datos con el Asistente de exportación y la importación de SQL Server
Los temas de esta sección muestran cómo conectarse a varios orígenes de datos utilizadas al ejecutar la importación de SQL Server y el Asistente para exportación. Tendrá que proporcionar información de conexión para los orígenes de datos en el **elegir un origen de datos** y **elegir un destino** páginas del asistente.

Los temas de esta sección describen cómo sólo a **conectarse a orígenes de datos** desde el **elegir un origen de datos** y **elegir un destino** páginas del asistente. Si está buscando algo, consulte [relacionados con tareas y contenido](#related).

## <a name="connect-to-other-data-providers"></a>Conectarse a otros proveedores de datos
Para obtener información sobre cómo conectarse a un origen de datos que no aparece aquí, consulte [la referencia de las cadenas de conexión](https://www.connectionstrings.com/). Este sitio de terceros contiene las cadenas de conexión de ejemplo y obtener más información acerca de los proveedores de datos y la información de conexión que se necesiten.

-   **Más información sobre cómo funciona el asistente.**

    -   **Más información sobre el asistente.** Si quiere obtener información general sobre el asistente, consulte [Asistente para importación y exportación de SQL Server](../../integration-services/import-export-data/import-and-export-data-with-the-sql-server-import-and-export-wizard.md).

    -   **Obtenga información acerca de los pasos del asistente.** Si desea obtener información acerca de los pasos del asistente, consulte [los pasos de la importación de SQL Server y el Asistente para exportación de](../../integration-services/import-export-data/steps-in-the-sql-server-import-and-export-wizard.md). -   **Inicie al asistente.** Si está listo para ejecutar el asistente y simplemente quiere saber cómo iniciarlo, consulte [Iniciar el Asistente para importación y exportación de SQL Server](../../integration-services/import-export-data/start-the-sql-server-import-and-export-wizard.md).

-   **Obtenga el asistente** Si desea ejecutar el asistente, pero no tiene [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] instalado en el equipo, puede instalar el Asistente para importación y exportación de [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] mediante la instalación de SQL Server Data Tools (SSDT). Para obtener más información, vea [Descargar SQL Server Data Tools (SSDT)](https://msdn.microsoft.com/library/mt204009.aspx).
