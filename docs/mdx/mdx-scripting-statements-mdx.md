# <a name="mdx-scripting-statements-mdx"></a>Instrucciones para scripting de MDX (MDX)
[!INCLUDE[ssas-appliesto-sqlas](../includes/ssas-appliesto-sqlas.md)]

  En las Expresiones multidimensionales (MDX), las siguientes instrucciones administran el contexto y el ámbito de la consulta, y el control del  flujo en los scripts de MDX.  
  
## <a name="in-this-section"></a>En esta sección  
  
|Tema|Descripción|  
|-----------|-----------------|  
|[Instrucción CALCULATE &#40;MDX&#41;](../mdx/mdx-scripting-calculate.md)|Calcula un subcubo y opcionalmente determina el orden de resolución de las dimensiones incluidas en el subcubo.|  
|[CASE (instrucción) &#40;MDX&#41;](../mdx/case-statement-mdx.md)|Permite obtener condicionalmente valores específicos de varias comparaciones.|  
|[EXISTING, palabra clave &#40;MDX&#41;](../analysis-services/multidimensional-models/mdx/mdx-query-existing-keyword.md)|Exige que un conjunto especificado se evalúe en el contexto actual.|  
|[Instrucción de inmovilización &#40;MDX&#41;](../mdx/mdx-scripting-freeze.md)|Bloquea los valores de celda de un subcubo especificado a sus valores actuales.|  
|[IF instrucción &#40;MDX&#41;](../mdx/mdx-scripting-if.md)|Ejecuta una instrucción si la condición es true.|  
|[Instrucción SCOPE &#40;MDX&#41;](../mdx/mdx-scripting-scope.md)|Limita el ámbito de las instrucciones MDX especificadas a un subcubo especificado.|
