# <a name="oracle-software-patches"></a>Revisiones de Software de Oracle
> [!IMPORTANT]  
>  Esta característica se quitará en una versión futura de Windows. Evite utilizar esta característica en nuevos trabajos de desarrollo y tenga previsto modificar las aplicaciones que actualmente la utilizan. En su lugar, utilice el controlador ODBC proporcionado por Oracle.  
  
 Revisiones para los productos de servidor de Oracle y su componente de cliente son necesarias para que funcione correctamente varios productos y tecnologías, como el controlador ODBC de Microsoft para Oracle, el proveedor Microsoft OLE DB para Oracle, la información de Internet de Microsoft Services (IIS), servicios de componentes (o Microsoft Transaction Server, si está utilizando Windows NT), y así sucesivamente.  
  
> [!NOTE]  
>  Las instrucciones siguientes pueden no sea completamente exacta porque el sitio FTP de Oracle está sujeta a cambios.  
  
### <a name="to-download-the-oracle-software-patches"></a>Para descargar las revisiones de software de Oracle  
  
1.  Conectarse al sitio FTP público en ftp.oracle.com de oracle. El identificador de usuario es "anónimo" y la contraseña es su dirección de correo electrónico.  
  
2.  Navegue hasta el siguiente directorio: /server/wgt_tech/server/windowsNT.  
  
3.  Para descargar revisiones más relevantes para Windows 95, Windows 98 y Windows NT, Windows 2000, desplácese hasta el subdirectorio para su versión de Oracle: 7.3 o 8.0. Los dos subdirectorios son /73patchsets y /80patchsets.  
  
4.  Para descargar revisiones para la tecnología de red de Oracle, ya sea SQL * Net o Net8, navegue hasta el siguiente directorio: / red.  
  
 Obtiene acceso a este sitio FTP desde el explorador Web no funcionen. Si tiene problemas, pruebe a usar a un cliente FTP "tradicionales" o usar el símbolo del sistema de DOS.  
  
> [!NOTE]  
>  Dado que Oracle corrige los errores en las versiones actuales y, a continuación, los retrofits a las versiones anteriores con revisiones de software, se recomienda descargar la revisión más reciente disponible. Esto es especialmente cierto para los componentes de cliente de servidor de Oracle. Si tiene preguntas acerca de cómo instalar estas revisiones, póngase en contacto con el soporte técnico de Oracle.
