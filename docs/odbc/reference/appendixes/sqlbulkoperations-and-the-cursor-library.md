# <a name="sqlbulkoperations-and-the-cursor-library"></a>SQLBulkOperations y biblioteca de cursores
> [!IMPORTANT]  
>  Esta característica se quitará en una versión futura de Windows. Evite usar esta característica en nuevos trabajos de desarrollo y piense en modificar las aplicaciones que actualmente utilizan esta característica. Microsoft recomienda usar la funcionalidad de cursor del controlador.  
  
 La biblioteca de cursores no se admite llamar a **SQLBulkOperations**. Para obtener información general sobre **SQLBulkOperations**, consulte [función SQLBulkOperations](../../../odbc/reference/syntax/sqlbulkoperations-function.md).
