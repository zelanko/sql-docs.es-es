# <a name="configuring-data-sources"></a>Configurar orígenes de datos
Información acerca de los orígenes de datos se almacena en el registro del sistema. Los usuarios modificar la información de origen de datos a través de un programa de administración. Esto puede ser el Administrador de ODBC, el dispositivo del Panel de Control de ODBC o un programa de administración escritos por un desarrollador de la aplicación o controlador.  
  
 Puede usar los comandos de PowerShell para modificar orígenes de datos. Para obtener más información acerca de estos comandos de PowerShell, consulte [comandos de PowerShell de Windows Data Access componentes](https://msdn.microsoft.com/library/windows/desktop/jj134064.aspx).  
  
> [!NOTE]  
>  En esta sección es específica para las plataformas Microsoft Windows®. Cómo se configuran los orígenes de datos en otras plataformas es específico de la plataforma.
