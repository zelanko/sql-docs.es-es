# <a name="storage-system-input-output-time-out"></a>Tiempo de espera de entrada y salida del sistema de almacenamiento
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] Esta regla comprueba el identificador EventId 9 se encuentra en el registro de eventos del sistema. Este mensaje indica que se ha superado el tiempo de espera de E/S en el sistema de almacenamiento.
