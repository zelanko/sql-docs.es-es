# <a name="database-mail-views-transact-sql"></a>Vistas del Correo electrónico de base de datos (Transact-SQL)
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]

  El Correo electrónico de base de datos tiene las siguientes vistas para visualizar contenido de correo electrónico propio, el estado de los mensajes y cualquier mensaje recibido, como los errores registrados por el Correo electrónico de base de datos. Estas vistas están en la base de datos **msdb** .

    :::column:::
