# <a name="sysstats_columns-transact-sql"></a>sys.stats_columns (Transact-SQL)
[!INCLUDE [sql-asdb-asdbmi-asa-pdw](../../includes/applies-to-version/sql-asdb-asdbmi-asa-pdw.md)]

  Contiene una fila para cada columna que forma parte de las estadísticas de **Sys. stats** .  
  
|Nombre de la columna|Tipo de datos|Descripción|  
|-----------------|---------------|-----------------|  
|**object_id**|**int**|Id. del objeto del que forma parte esta columna.|  
|**stats_id**|**int**|Id. de las estadísticas de las que forma parte esta columna.<br /><br />Si las estadísticas corresponden a un índice, el valor *stats_id* es el mismo que el valor *index_id* de la vista de catálogo [Sys. Indexes](../../relational-databases/system-catalog-views/sys-indexes-transact-sql.md) .|  
|**stats_column_id**|**int**|Ordinal de base 1 del conjunto de columnas de estadísticas.|  
|**column_id**|**int**|IDENTIFICADOR de la columna de **Sys. Columns**.|
