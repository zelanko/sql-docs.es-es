# <a name="propertystrvalue-property-sqlserviceadvancedproperty-class"></a>Propiedad PropertyStrValue (clase SqlServiceAdvancedProperty)
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]
  Obtiene el valor de cadena de la propiedad avanzada.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
object.PropertyStrValue [= value]  
```  
  
## <a name="parts"></a>Partes  
 *objeto*  
 A [clase SqlServiceAdvancedProperty](../../../relational-databases/wmi-provider-configuration-classes/sqlserviceadvancedproperty-class/sqlserviceadvancedproperty-class.md) objeto que representa una propiedad avanzada.  
  
## <a name="property-valuereturn-value"></a>Valor de propiedad y valor devuelto  
 Valor de cadena que especifica el valor de la propiedad avanzada.
