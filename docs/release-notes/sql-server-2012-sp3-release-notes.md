# <a name="sql-server-2012-sp3-release-notes"></a>Notas de la versión de SQL Server 2012 SP3
Estas notas de la versión se describen los problemas que debe conocer antes de instalar o solucionar problemas de SQL Server 2012 Service Pack 3. Las notas de la versión solo están disponibles en Internet, no se incluyen en el disco de instalación. Se actualizan periódicamente, conforme se van detectando nuevos problemas. Para obtener una lista o mejoras y correcciones, consulte [información de la versión de SQL Server 2012 Service Pack 3](http://support.microsoft.com/help/3072779/sql-server-2012-service-pack-3-release-information) para obtener más información.  
  
## <a name="choose-the-correct-file-to-download-and-install"></a>Elegir el archivo correcto para su descarga e instalación  
Use la tabla siguiente para determinar la ubicación y el nombre del archivo que debe descargar en función de la versión que tenga instalada actualmente. Las páginas de descarga contienen los requisitos del sistema e instrucciones básicas para la instalación.  

##<a name="download-pages"></a>Páginas de descarga
El vínculo siguiente para el método main descargar los paquetes de SQL Server 2012 SP3.
- [SP3 de SQL Server 2012](http://go.microsoft.com/fwlink/?linkid=615935)
- [SP3 SQL Server 2012 Express](http://go.microsoft.com/fwlink/?linkid=692144)
- [Feature Pack de Microsoft SQL Server 2012 SP3](http://go.microsoft.com/fwlink/?linkid=615941)

Para obtener más información para identificar la ubicación y el nombre del archivo que desea descargar en función de la versión instalada actualmente, vea la sección "Seleccionar el archivo correcto para descargar" en [información de la versión de SQL Server 2012 Service Pack 3](https://support.microsoft.com/en-us/help/3072779/sql-server-2012-service-pack-3-release-information).
