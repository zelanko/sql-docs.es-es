# <a name="msreportserverinstance-properties---editionid"></a>Propiedades de MSReportServer_Instance: EditionID
  Devuelve la edición del servidor de informes. Solo lectura.  
  
## <a name="syntax"></a>Sintaxis  
  
```vb  
Public Dim EditionID As UInt  
```  
  
```csharp  
public uint EditionID;  
```  
  
## <a name="property-value"></a>Valor de la propiedad  
 Una **uint** que contiene la edición del servidor de informes.
