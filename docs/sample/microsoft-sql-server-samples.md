## <a name="new-wide-world-importers-sample-database"></a>¡¡Nuevo!! Base de datos de ejemplo de Wide World Importers

Nos complace ofrecer una nueva base de datos de ejemplo denominada Wide World Importers (WWI), desarrollado para el diseño de la base de datos de presentación mediante características actuales, herramientas y técnicas de diseño. Los ejemplos incluyen un OLTP y una base de datos OLAP que se generan para que se ejecute en SQL Server 2016 y versiones posteriores.

## <a name="sql-server-samples-repository"></a>Repositorio de ejemplos de SQL Server
Los ejemplos de código se encuentran en el [Microsoft SQL Server Samples](https://github.com/microsoft/sql-server-samples) repositorio de GitHub. Cada ejemplo incluye un archivo Léame que explica cómo ejecutar y usar el ejemplo. 

## <a name="working-in-github"></a>Trabajar en GitHub
Para trabajar en GitHub, vaya a https://github.com/microsoft/sql-server-samples y bifurcar el repositorio. Trabajar en su propia bifurcación y cuando esté listo para enviar para realizar un cambio o publicar su ejemplo por primera vez, enviar una solicitud de incorporación de cambios en la bifurcación principal de ejemplos de sql server. Uno de los aprobadores se revise su solicitud y Aceptar o rechazar la solicitud de extracción.
