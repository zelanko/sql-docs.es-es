# <a name="object-already-exists"></a>El objeto ya existe
[!INCLUDE[SQL Server Azure SQL Database Synapse Analytics PDW ](../../includes/applies-to-version/sql-asdb-asdbmi-asa-pdw.md)]
El objeto que intenta guardar tiene el mismo nombre que otro objeto de la base de datos. Elija otro nombre y continúe.  
  
También puede indicar que el nombre de columna ya existe.
