# <a name="rename-a-database-diagram-visual-database-tools"></a>Cambiar el nombre de un diagrama de base de datos (Visual Database Tools)
Puede cambiar el nombre de un diagrama de base de datos en el Explorador de objetos. Requiere el permiso ALTER en la base de datos.  
  
### <a name="to-rename-a-database-diagram"></a>Para cambiar el nombre de un diagrama de base de datos  
  
1.  En el Explorador de objetos, expanda la carpeta **Diagramas de base de datos** .  
  
2.  Haga clic con el botón secundario en el diagrama de base de datos cuyo nombre desea cambiar.  
  
3.  Elija **Cambiar nombre** en el menú contextual.  
  
    Se abre un cuadro de edición alrededor del nombre del diagrama en el Explorador de objetos.  
  
4.  Especifique el nuevo nombre del diagrama de base de datos y presione **Entrar**.  
  
    El diagrama de base de datos aparece en el Explorador de objetos con el nuevo nombre.
