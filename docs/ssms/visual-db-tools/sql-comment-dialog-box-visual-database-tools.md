# <a name="sql-comment-dialog-box-visual-database-tools"></a>Comentario de SQL (cuadro de diálogo, Visual Database Tools)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)] En este cuadro de diálogo se proporciona un área modificable en la que se puede escribir una descripción detallada de la consulta. Para obtener acceso a este cuadro de diálogo, abra la consulta o la vista en el Diseñador de consultas y vistas y, en la ventana **Propiedades** , seleccione **Comentario SQL**y haga clic en los puntos suspensivos (**...**) que aparecen a la derecha del campo de propiedad.
