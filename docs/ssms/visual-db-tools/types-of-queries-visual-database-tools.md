# <a name="types-of-queries-visual-database-tools"></a>Tipos de consultas (Visual Database Tools)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)] El Diseñador de consultas y vistas proporciona características integradas para algunos tipos de consulta (de selección, inserción de resultados, inserción de valores, actualización, eliminación y creación de tabla) y admite también las consultas de unión, búsqueda de texto completo y de expresiones CASE.
