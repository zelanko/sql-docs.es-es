# <a name="mssql-cli-command-line-query-tool-for-sql-server"></a>herramienta de línea de comandos de consulta MSSQL-cli para SQL Server
[!INCLUDE[appliesto-ss-asdb-asdw-xxx-md](../includes/appliesto-ss-asdb-asdw-xxx-md.md)]

MSSQL-cli es una herramienta de línea de comandos interactiva para realizar consultas en SQL Server, instalarlo en Windows, macOS o Linux.

## <a name="install-mssql-cli"></a>Instalación de mssql-cli

Para obtener instrucciones detalladas, consulte el [Guía de instalación](https://github.com/dbcli/mssql-cli/blob/master/doc/installation_guide.md), o si sabe pip, instalación, ejecute el comando siguiente:

```$ pip install mssql-cli```
